## スコープ外の機能

以下の機能は本プロジェクトでは**実装しない**。詳細は [requirements.md](requirements.md) のスコープ外セクションを参照。

- インプレッション数表示
- リツイート / 引用ポスト
- ダイレクトメッセージ（DM）
- 通知機能（プッシュ通知・リアルタイム通知）
- ハッシュタグ・メンション機能
- 管理者画面
